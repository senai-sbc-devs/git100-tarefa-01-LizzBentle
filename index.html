Git Workflow

Esta página tem como objetivo explicar os passos básicos do fluxo de trabalho no Git.

Clone

Clone um repositório remoto para sua máquina local.

Exemplo de comando Git: git clone URL_do_repositorio

Branch

Crie e gerencie branches para trabalhar em diferentes funcionalidades.

Exemplo de comando Git: git branch nome_do_branch

Commit

Registre as mudanças no repositório.

Exemplo de comando Git: git commit -m "Mensagem do commit"

Merge

Combine as alterações de um branch ao branch principal.

Pull Request

Solicite a incorporação das mudanças feitas em uma branch para o branch principal.
